#### Prerequisite

```License file``` and ```Serial Number``` for SigmaPlot v12.0 is required.

#### SigmaPlot v12.0

1. Run "SPW_WebInstall_Please_Run_as_Administrator.exe" as Administrator
2. Select <YOUR_LICENSE_FILE> as License File
3. Type <YOUR SERIAL NUMBER> as serial number
4. SigmaPlot will be installed at:
   1. System Executable Files
      1. C:\Program Files (x86)\SigmaPlot\SPW12\Spw.exe
   2. User Configuration Files
      1. C:\Users\<YOUR_USER_NAME>\Documents\SigmaPlot\SPW12\

#### SigmaPlotConfig Folder

1. Extract the ```SigmaPlotConfig - Extract and Place it under Documents.zip``` and copy the ```SigmaPlot``` folder to ```C:\Users\<YOUR_USER_NAME>\Documents\SigmaPlot\SPW12\```
